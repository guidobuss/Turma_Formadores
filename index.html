Glossário de Robótica

Um glossário é uma lista em ordem alfabética de palavras relacionadas a robótica e termos específicos de uma área de conhecimento, acompanhados de suas respectivas definições. Ele funciona como um minidicionário temático, facilitando a compreensão de vocabulários técnicos, regionais ou de idiomas estrangeiros.

Módulo ultrassônico

O módulo ultrassônico HC-SR04 é um sensor de distância eletrônico que emite ondas sonoras inaudíveis (ultrassom) a 40 kHz e mede o tempo que o eco leva para refletir de volta após atingir um objeto. Utiliza o mesmo princípio de ecolocalização dos morcegos para calcular a distância com alta precisão.

Módulo Sensor IR

O módulo sensor infravermelho (IR) para Arduino é um dispositivo que utiliza luz infravermelha para detectar obstáculos, seguir linhas ou receber comandos à distância. Existem duas versões principais: Sensor de Obstáculo/Reflexivo e Receptor IR
